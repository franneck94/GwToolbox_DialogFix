# EMO Helper

The window will only be rendered if you are on an Rit/R or D/R.

![alt](../imgs/EMO.png)

## TLDR

- In Full-man Teams:
    - This feature will let you go fully afk at dhuum fight
- In Low-man Teams:
    - You can go fully afk for the whole run :)

## Description

- DB Routine:
    - Kill Chamber and Basement Skele (*)
    - Kill Skeles at Vale House (*)
    - Kill Vale Spirits Groups (*)
    - Smart Skill Routine At Dhuum Fight (you can go fully afk)

- Move-To Feature:
    - Pre-defined set of positions, like goto Lab Reaper, etc.
    - Mimicking the positions and movements of a real player (*)
    - Take Restore and Escort at Lab Reaper (*)


\* Only for low-man tactics
